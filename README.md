# Дашборд продуктовых метрик · Apache Superset

Интерактивный дашборд **«AZOT. Основные метрики»** для мониторинга ключевых продуктовых и финансовых показателей сервиса. Собран в Apache Superset на учебных данных: финансы, аудитория, источники трафика, покупки, юнит-экономика и воронка — в динамике и в разрезах, с фильтрами, пересчитывающими весь дашборд.

🔗 **Дашборд:** https://superset-edu.tbank.ru/superset/dashboard/p/1v3gJpVXmRr/

> Дашборд расположен в учебной среде и открывается после авторизации. Чтобы результат был виден без доступа, ниже приведены скриншоты.

## Что на дашборде

- **Финансы** — выручка и её динамика по месяцам.
- **Пользователи** — DAU / WAU / MAU и их динамика во времени.
- **Источники трафика** — распределение сессий по каналам (organic search, paid search, social, direct, affiliate, email, referral).
- **Операционные системы** — доли платформ (Android, iOS, Windows, macOS, Linux).
- **Покупки** — количество покупок, средний чек.
- **Юнит-экономика** — ARPU и ARPPU, в том числе по неделям.
- **Воронка покупок** — переходы Каталог → Корзина → Покупка по доле сессий.

**Фильтры:** период, грануляция, операционная система, источник трафика.
